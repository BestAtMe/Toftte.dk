About the project

Simpltronics is an online learning platform startup that aims at making leaning mechatronics easier. Mechatronics is the craft of combining coding with mechanics to create for example robots. The startup consisted of myself and 4 others.

The website can be accessed here: simpltronics.netlify.app/

The website was motivated by the lack of modern mechatronic teaching material available. Many universities in the world still teach mechatronics via a textbook which is pages upon pages of text. We thought that this is an outdated way of teaching and wanted to give beginners a place where they would learn not only by reading but by interacting and looking at the teaching material.

The startup is dissolved since the team had other interests they wanted to pursue instead.

The process

The website has a lot of mechatronics resources gathered as well as multiple guides on how to get started, how to code, how to 3D print as well as other guides on different projects. This gives new users an easy overview on how to get started. The projects would have different skill levels so that the user could progress in skill and keep challenging themselves.

Another great feature of the website is all of the interactive learning that is implemented. An example of this is the interactive coding tool that was implemented that uses a block structure to give the user an easy overview of what features change what code, and how the different variables and components of the circuit can interact with the code.

The validation of the website happened in the form of multiple workshops. In the span of 5 workshops more than 40 participants used the website to create a beginner project. Data from the participants were collected via a questionnaire after each workshop. The feedback was overall very positive and the website succeeded in providing a more modern way of teaching mechatronics.
